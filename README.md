# Scrum AI Agent UI

Two-screen interface for a Scrum specialist AI agent:

- Chat and PDF submission screen for project descriptions.
- Planning dashboard to edit sprints, tasks, and metrics.

## Run

1) Create and activate a virtual environment

```powershell
python -m venv .venv
.venv\Scripts\activate
```

2) Install dependencies

```powershell
pip install -r requirements.txt
```

3) Start the server

```powershell
python app.py
```

Open http://127.0.0.1:5000
